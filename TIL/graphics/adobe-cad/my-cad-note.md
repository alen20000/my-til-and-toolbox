### env setting
---
* `LA` 	:	圖層選單
* `SE` 	:	草圖設置介面。 至少對象捕捉，左側都打開
* `OP` 	:	客製化界面
* `NEW`	:	圖紙設置,選"acadiso" 國際公制
		畫一條300長的線，這就是A4紙的標準<br>
### quick order
---
* `REC`	:	矩形。refere : @100,100 - 畫 長寬各100的矩形
* `DIM`	:	水平標記
* `DAL`	:	對齊標註
* `ML`	:	多線指令
* `CRTL`+`Y`:	取消上次的撤銷，等於回來了。
* `DIM` :	標註工具
* `DAN` :	弧長標註
* `dimspace`:	對齊標註排序	
* `O`	:	偏移。做圖常用，配合`x`大部分解快速做內側物件輪廓
* `BR`	:	打斷長線。可`BR`後街`F`做到精準切邊
* `J`	:	連線合併。
* `SPLINE`:	描輪廓
* `TR`	:	剪切工具。
* `ARC` :	畫弧
* `SC`	:	設置基點，把物件放大縮小。
* `arrary`	:複製多物件，1.選要複製對象 2.輸入 po 3. 選擇中心點 4. 點選項目設置數量<br>
			arrary的物件被當作一個整體，要`tr刪除的話，需先用`x`把物件分開
* `move`:	移動物件。[!]第一點是設定要移動物件的基點	
* `EX`:		延伸指令。`TR`修剪指令，能跟`EX`交互，多按`shift`轉換	
* `FILLET`:	倒圓角指令。接R改半徑圓角；接P能改多段線批量圓角。
### quick key
---
* `F3`	:	長期開啟。對象捕捉。syn:SE 可以設置
* `F7`	:	預設關掉。柵格。
* `F8`	:	預設關掉。精準畫會歪，畫不精準圖可以用。
* `F9`	:	預設關掉。捕捉模式 (Snap Mode)，工具會被鎖在網格的感覺，很爛。
* `F10`	:	長期開啟。
* `F11`	:	長期開啟。物件鎖點追蹤 (Object Snap Tracking) —— 「幾何推導」
* `F12`	:	切換動態輸入的開關，才能用數值輸入，打開會卡頓，默認不開
* `D`	:	設定標註樣式
* `VPORTS` :	設定多畫面

## 特定操作
- 線寬調整:`crtl`+`1`。調整線粗之後，輸入`LWDISPLAY`
- 元件操作:按`B`，可設定為Block，搭配GCEN or M2P 精準抓出物件中心基點<br>輸入`insert`能調出元件視窗
開窗口畫圖:視圖(v) -> 雙口 -> 選擇

- 做彎曲線:圖紙寫R8之類的，先按F，然後輸入8，點要彎的區域。

 -F11虛線定位技術 : 要會利用虛線的定位功能配合精準距離快速布置，不要傻傻拉

 -TK連續追蹤:用TK輔助功能達到快速定位。舉例，按下工具`c`，不要點滑鼠定位，先輸入'tk'

 -直線拉角度 : 輸入`line`，再輸入`<`，後輸入數值，就能得到角度斜線
