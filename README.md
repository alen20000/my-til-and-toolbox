### 🛠 我的工具箱

*   **webp-to-mp4**: 圖片 webp 轉 mp4 影片檔。 (依賴 ffmpeg, python)
*   **mp4-quickly-merge**: 批量 mp4 合併單一檔案。
    *   依賴 ffmpeg, python 預設快速合併模式。
    *   腳本下有另套 CPU 編碼 pattern 可以選擇，但 GPU 調用需再調參數。
*   **md_list_generator**: 配合 github 表單用的 md 連結格式生成器。
